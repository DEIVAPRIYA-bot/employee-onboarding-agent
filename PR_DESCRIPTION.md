Title: Add KBC Onboarding Assistant scaffold (Flask app, SharePoint client, agent)

Adds an initial Flask-based KBC Onboarding Assistant scaffold that uses Microsoft Graph / SharePoint as the primary source of truth. Includes endpoints POST /query and GET /status/<employee_email>, a minimal SharePoint client, agent logic, and usage docs (config.example.env). Intended as a starter you can extend and secure for production.
